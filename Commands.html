Commands

NappyTube is incredibly easy to use

There are only a few commands to remember to begin playing music.

Command Arguments Example Output
!listen (Playlist Name) !listen top 200 Listen to playlist: Top 200
!leave none !leave Leave voice channel
!info none !info Shows what's playing
!dev none !dev Learn about the developer
